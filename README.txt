###### Instruções de execução e compatibilidade ######

Basta abrir o arquivo index.html no seu Browser favorito

ATENÇÃO¹:
Todos os arquivos e pastas devem ser salvos no mesmo local, preservando a estrutura de diretórios do commit mais recente.

ATENÇÃO²:
Embora em tese o programa seja suportado pela maioria dos Browsers recentes, o mesmo foi testado apenas nos Browsers Mozilla Firefox, Google Chrome, Chromium e Opera.
Para mais detalhes por favor clique no botão "Sobre" na tela inicial do programa.
